--- PyForge (2D и 3D движок/фраемворк) --- 
 
Данный движок нужен для создание 3D и 2D игр, также используется для создание графической оболочки. 
Преимущество: 
    Создание Окна и рабочей области в пару клико 
    Упаковка всех зависимостей в один фаил (pyz или exe) 
    Использование OpenGL 
 
Для работы: 
    pip3 insatall pygame, numpy 
Для полной работы: 
    pip3 insatall PyOpenGL 
 
Для защиты пайтон скриптов от злома лучше использовать: pyarmor pack -e "--onefile" скрипт.py
 
----------- Файлы и папки: 
 
Фаил main.py: 
    нужен для создания проекта с использованием движка 
 
Фаил update.py: 
    нужен для обновления движка на новую версию 
 
Фаил info.py: 
    в этом файле хранится информация о движке(версия, и тд) 
    version = '0.0.1' 
    logotip = "PyForge\\image\\logotip\\20250928_1928_image.png" 
 
Фаил init.py: 
    в этом файле импортируется все другие файлы этого движка 
    def is_admin() -> bool #проверяет запущенна ли программа от имени администратора 
     
Фаил game_obgect.py: 
    в нём хранится класс Game для построение движка(выполняется в основном процессе) 
    
 
Папка pygames: 
    в ней хранятся инструменты для рисования, создание объектов с использованием pygame и вычисляемые на процессоре. 
 
Папка OpenGL: 
    в ней хранятся инструменты для рисования и создать объектов с использованием OpenGL (видеокарты), рекомендуется для 3D объектов с текстурами.
